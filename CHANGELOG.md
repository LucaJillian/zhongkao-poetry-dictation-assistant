# 古诗背诵助手 - 导入功能重构摘要

## 修改日期
2026-05-18

## 主要更新

### 1. 修复批量导入换行问题
**问题：** 原导入算法按 `\n` 分割文本，导致诗文中的换行被错误分割

**解决方案：**
- 新增支持 `---` 或 `===` 作为古诗记录分隔符
- 支持 `Title:/Author:/Content:/Notes:` 格式的多行解析
- 保持向后兼容，仍支持旧版 `|||` 一行一首格式

### 2. 新增单首导入功能
**UI：** 在导入管理页面添加选项卡切换
- 批量导入（原有功能增强）
- 单首导入（新增表单）

**单首导入表单字段：**
- 标题（必填）
- 作者（可选）
- 诗文（必填，支持多行）
- 注释（可选）

### 3. 导入算法重构

**新格式示例：**
```
Title: 静夜思
Author: 李白
Content: 床前明月光，疑是地上霜。
举头望明月，低头思故乡。
Notes: 【注释】①霜：白霜。

---
Title: 春晓
Author: 孟浩然
Content: 春眠不觉晓，处处闻啼鸟。
夜来风雨声，花落知多少。
Notes:

（每首诗之间用 --- 分隔）
```

**旧格式仍支持：**
```
静夜思|||李白|||床前明月光，疑是地上霜。举头望明月，低头思故乡。|||【注释】①霜：白霜。
```

### 4. UI改进
- 添加导入选项卡样式（CSS）
- 优化导入页面布局
- 添加清空表单功能

## 技术细节
- 使用正则表达式 `\n-{3,}\s*\n|\n={3,}\s*\n` 分割古诗记录
- 逐行解析字段标签，支持多行续行
- 自动识别新旧格式并选择对应解析算法
